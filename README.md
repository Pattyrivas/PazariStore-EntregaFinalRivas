# 🛍️ Pazari Store

E-commerce de accesorios desarrollado como proyecto final del curso de Desarrollo Frontend en Coderhouse. Permite explorar un catálogo de productos, agregarlos a un carrito de compra y simular el proceso de checkout.

🔗 **Demo en vivo:** https://pazaristore-deploy.vercel.app

## ✨ Funcionalidades

- Catálogo de productos con datos dinámicos
- Carrito de compra (agregar, eliminar, actualizar cantidades)
- Cálculo automático de totales
- Diseño responsive (mobile-first)
- Componentes reutilizables en React

## 🛠️ Stack Tecnológico

- **React** — librería principal de UI
- **Vite** — bundler y entorno de desarrollo
- **Tailwind CSS** — estilos utilitarios y diseño responsive
- **JavaScript (ES6+)**

## 🚀 Cómo ejecutar el proyecto localmente

```bash
# Clonar el repositorio
git clone https://github.com/Pattyrivas/PazariStore-EntregaFinalRivas.git

# Entrar a la carpeta
cd PazariStore-EntregaFinalRivas

# Instalar dependencias
npm install

# Levantar el servidor de desarrollo
npm run dev
```

El proyecto quedará disponible en `http://localhost:5173`.
